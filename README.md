## git командууд
git командуудын тайлбар, тусламжийн репо

git эхнээс нь дуустал гэдэг сургалт дээр үзсэн командууд болон сурагчдын оролцоотойгоор хөгжүүлэгдэж байгаа репо юм

```javascript
function test() {
console.log("look ma’, no spaces");
}
```

## Командуудын жагсаалт

'git add *' **бүх шинэ болон өөрчлөгдсөн файлуудыг стайжилнэ**

'git add .' **бүх шинэ болон өөрчлөгдсөн файлуудыг стайжилнэ**

'git add -A' **бүх шинэ болон өөрчлөгдсөн файлуудыг стайжилнэ**

'git add [file.txt]' **file.txt файлыг стайжилнэ**
